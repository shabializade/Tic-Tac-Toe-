
# Tic-Tac-Toe Game in C

Console-based Tic-Tac-Toe game with dynamic board size and advanced features.

## Features:
- Board size selection (3x3, 4x4, 5x5)
- Two-player gameplay
- Queue-based move system (for 3x3 board)
- Automatic removal of oldest moves (FIFO logic)
- Game logging with timestamps
- Win detection (rows, columns, diagonals)

## Technologies:
- C programming language
- Dynamic memory allocation (malloc)
- File handling for logging

## How to Run:
1. Compile:
   gcc tictactoe.c -o game
2. Run:
   ./game
